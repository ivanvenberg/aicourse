## Установка до первой встречи

### 1. Скачай Cursor

Cursor — это редактор кода с AI внутри. Именно в нём мы будем всё делать.

👉 **[cursor.com/download](https://cursor.com/download)**

Скачай, установи, запусти.

### 2. Создай аккаунт в Cursor

При первом запуске попросит войти. Нажми **Sign Up** и зарегистрируйся — это бесплатно.

> Бесплатного плана хватит для всего курса. Если хочешь больше — есть Pro за $20/мес, но это не обязательно.

### 3. Скачай этот репозиторий

Нажми зелёную кнопку **Code → Download ZIP** вверху страницы.

Распакуй архив в удобное место — например, на рабочий стол.

### 4. Открой папку в Cursor

- Запусти Cursor
- Нажми **File → Open Folder**
- Выбери папку `ai-course` (которую только что распаковал)

### 5. Проверь, что всё работает

Открой любой файл в Cursor и нажми:
- **Mac:** `Cmd + K`
- **Windows/Linux:** `Ctrl + K`

Должна появиться строка для AI-запроса. Если появилась — всё готово! 🎉
