## Задание 2. MCP для твоего инструмента

Выбери MCP под свою профессию (см. theory.md) и установи:

```bash
claude mcp add figma    # для дизайнеров и моушн
claude mcp add gdrive   # для главредов
claude mcp add gsheets  # для СМО, CRM, архитектора
claude mcp add gmail    # для СМО, CRM маркетолога
```

После установки попроси Claude что-то сделать через этот инструмент:
```
Открой мой последний файл в [Figma/Notion/Google Sheets] и опиши что там есть.
```

✅ MCP работает — написал плюс в чат

## Задание 4. Команда агентов — дебаты

Сначала включи экспериментальные команды агентов:

```
Добавь в настройки Claude Code переменную:
CLAUDE_CODE_EXPERIMENTAL_AGENT_TEAMS=1
```

Перезапусти Claude Code. Теперь создаём команду дебатёров:

```
Создай команду агентов для брейншторма идей по моему продукту:
- Агент Генератор: читает prd.md и предлагает 3 идеи для улучшения
- Агент Адвокат: берёт каждую идею и объясняет почему она хорошая
- Агент Критик: берёт каждую идею и объясняет почему она плохая
В конце — вывод: какую идею брать в работу

Сохрани команду агентов в .claude/agents/
```

✅ Команда создана и запущена — написал плюс в чат

## Задание 5. Контент-пайплайн (особенно для главредов)

Если ты создаёшь контент — собираем автоматическую систему:

**Шаг 5.1** — Создаём скилл tone-of-voice:
```
Прочитай мои последние 10 текстов (или опиши мой стиль голосом).
Создай скилл tone-of-voice который содержит инструкции как писать в моём стиле.
Сохрани в .claude/skills/tone-of-voice/skill.md
```

**Шаг 5.2** — Создаём скилл find-news:
```
Создай скилл find-news который:
1. Открывает браузер
2. Заходит на [твои любимые источники]
3. Находит 5-7 новостей релевантных моей теме: [опиши тему]
4. Сохраняет список в news-today.md
```

**Шаг 5.3** — Запускаем пайплайн:
```
/find-news
```
Потом:
```
На основе news-today.md и скилла /tone-of-voice
напиши 3 поста в моём стиле.
Каждый пост сохрани отдельным файлом в posts/
```

✅ Посты написаны в твоём стиле — написал плюс в чат
