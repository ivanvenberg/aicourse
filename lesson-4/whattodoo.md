## 🔥 Упражнение по комнатам: «Битва агентов»

*Практика из главы 3: Plan-and-Execute vs ReAct, мульти-агентность.*

Две группы. Обе работают с одним PRD (берём чей-то из группы).

**Группа А — Plan-and-Execute:**
```
Составь план анализа этого продукта: что нужно исследовать.
После того как план готов — выполни каждый пункт.
```

**Группа Б — ReAct с субагентами:**
```
Запусти 3 субагента параллельно:
1. Найди сильные стороны продукта
2. Найди слабые стороны  
3. Предложи 5 улучшений
Каждый сохраняет в отдельный файл. После — объедини в summary.md
```

Сравниваем:
- Чьи инсайты глубже?
- Кто нашёл что-то что другие пропустили?
- Какой подход занял меньше токенов?

✅ Обе группы поделились результатами
