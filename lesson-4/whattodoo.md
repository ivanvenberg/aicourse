# Урок 4 — на встрече

**Цель:** `brief.md` + ссылка с лендингом и одной платной фичей + скилл.

Теория: `theory.md`. Шаблон: `brief-template.md`.

---

## 0. Фидбек (10 мин)

По кругу, 1 мин: что показал после урока 3, один инсайт от пользователя.

---

## 1. Выбор продукта (25 мин)

1. Новый репозиторий **или** папка `my-product/` — не путать с учебным чатом  
2. Скопируй `brief-template.md` → `brief.md`  
3. Нет идеи → `ideas-fallback.md`  
4. Сообщение знакомому:

```
Делаю [фраза из brief]. Заплатил бы / взял бы на работу? Честно.
```

5. Ответ — в «Доказательство»

✅ `brief.md` + цитата (или «отправил, жду») → **+ в чат**

---

## 🔥 «60 секунд за деньги»

В парах. Один — покупатель, один — фаундер.

60 секунд **только** из brief: кому, боль, цена. Слова «AI», «нейросеть», «курс» — запрещены.

Покупатель: «купил бы» / «не понял за что плачу» / «дорого, потому что…»

Меняетесь.

---

## 2. Лендинг + app (25 мин)

```
Прочитай brief.md.
Один проект, один деплой Vercel:
- / — лендинг: кому, боль, что получит, цена, кнопка
- /app — заглушка под платную фичу
Создай CLAUDE.md и MEMORY.md из brief.
```

✅ Ссылка открывается → **+ в чат**

---

## 3. Платная фича (25 мин)

```
Реализуй в /app ОДНУ фичу из brief — ту, за которую цена.
Supabase если нужно хранить данные.
Путь с лендинга → результат < 2 минут.
```

✅ Работает по ссылке → **+ в чат**

---

## 4. Скилл (15 мин)

```
Какая рутина вокруг этого продукта каждую неделю?
Скилл в .claude/skills/ — одна команда.
Проверь: /команда запускается.
```

✅ Один прогон на встрече → **+ в чат**

---

## 5. MCP (15 мин)

Подключи один MCP под профессию. Одно действие **для продукта** из brief.

Не встало → в домашку, не стопорим группу.

---

## 🔥 «Битва агентов»

Один `brief.md` на комнату (с разрешения автора).

**Команда А — план:** улучшить платную фичу по шагам → `team-a.md`

**Команда Б — 3 субагента:**
1. Что сломает покупатель за 5 минут  
2. Что добавить, чтобы заплатили ×2  
3. Одна правка UI за 10 минут кода  

→ `summary.md`

Сравни: чьи идеи **продают**, не «красивее».
