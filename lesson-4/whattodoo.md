## 2. Агенты челленджат идею (25 мин)

### Установка агентов (5 мин)

Скопируй из `agents-templates/` в свой проект:

```
.claude/agents/builder.md
.claude/agents/skeptic.md
.claude/agents/user-voice.md
```

Или: `/agents` → Create → Project.

### Запуск (20 мин)

```
Прочитай project.md.

Параллельно запусти субагентов: builder, skeptic, user-voice.
Каждый пишет в agents/ как в его инструкции.

Потом ты — судья. Создай agents/verdict.md:
- GO / PIVOT / KILL
- MVP на сегодня (одна вещь)
- что выкинуть из scope
- одна фраза для демо через 2 недели
```

🔥 **Доброволец** читает вслух только `verdict.md` — 60 сек.

✅ `agents/verdict.md` → **+ в чат**

## 3. MVP сегодня (30 мин)

По `verdict.md` — **одно** до конца созвона:

| Тип | MVP сегодня |
|-----|-------------|
| Контент | `style.md` + скилл `/post` + один файл в `posts/` |
| Бот | `knowledge/faq.md` + первый ответ (можно локально) |
| Веб | лендинг + `/app` с одной кнопкой |

```
По agents/verdict.md сделай MVP из project.md.
Только то, что в verdict. Не больше.
Создай CLAUDE.md из project.md.
Покажи за 30 секунд — ссылка или локально.
```

✅ ссылка / скрин / демо 30 сек → **+ в чат**
