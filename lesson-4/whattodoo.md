## 🔥 Упражнение по комнатам: «Битва агентов»

Делимся на две группы. Обе работают с одним и тем же продуктом (берём чей-то prd.md).

**Группа А:**
```
Запусти 3 субагента параллельно:
1. Найди главные сильные стороны продукта
2. Найди главные слабые стороны
3. Предложи 5 идей для улучшения
Каждый сохраняет результат в analysis/
```

**Группа Б:**
Делает то же самое но добавляет:
```
...после того как все три агента закончат —
сведи результаты в один файл summary.md
с итоговыми рекомендациями
```

**Сравниваем результаты:**
- Чьи инсайты лучше?
- Что нашла группа Б что не нашла группа А?
- Как формулировка промпта влияет на качество?

✅ Обе группы поделились результатами → обсудили

## Задание 3. Команда дебатёров (15 мин)

Сначала включаем:
```
Добавь в настройки Claude Code переменную:
CLAUDE_CODE_EXPERIMENTAL_AGENT_TEAMS=1
```

Перезапускаем Claude Code. Потом:

```
Создай команду агентов для брейншторма:
- Генератор: читает prd.md и предлагает 3 идеи для улучшения
- Адвокат: берёт каждую идею и объясняет почему она хорошая
- Критик: берёт каждую идею и объясняет почему она плохая
Вывод: какую идею реализовать первой и почему.
Сохрани команду в .claude/agents/
```

✅ Команда создана и запущена → написал + в чат

## Задание 4. Контент-пайплайн (для главредов)

**Шаг 1 — tone-of-voice:**
```
Прочитай мои последние 10 текстов [или: я опишу свой стиль голосом].
Создай скилл tone-of-voice с инструкциями как писать в моём стиле.
Сохрани в .claude/skills/tone-of-voice/skill.md
```

**Шаг 2 — find-news:**
```
Создай скилл find-news который:
1. Открывает браузер
2. Заходит на [перечисли свои источники]
3. Находит 5-7 новостей по теме: [твоя тема]
4. Сохраняет список в news-today.md
```

**Шаг 3 — запускаем:**
```
/find-news
```
Потом:
```
На основе news-today.md и скилла /tone-of-voice
напиши 3 поста в моём стиле.
Каждый пост — в отдельный файл в posts/
```

✅ Три поста написаны в твоём стиле → показал группе
