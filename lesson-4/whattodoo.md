## 0. Фидбек (10 мин)

## 1. Проект (20 мин)

1. Репозиторий **или** папка `my-project/` — не учебный чат  
2. `project-template.md` → `project.md`  
3. Нет идеи → `ideas-fallback.md`  
4. Сообщение знакомому:

```
Делаю [фраза из project]. Стало бы тебе проще, если …? Честно, 2 предложения.
```

5. Цитата — в раздел «Проверка с человеком»

✅ `project.md` → **+ в чат**

---

## 2. Perplexity / Parallel (25 мин)

```bash
cd examples/research
python3 -m venv .venv && source .venv/bin/activate
pip install -r requirements.txt
# ключи в .env (docs/env.example)
python ask_perplexity.py "твоя тема из project.md"
python parallel_search.py "твоя тема"
```

Агенту:

```
Возьми output/*.json из examples/research.
Собери research/sources.md — только выдержки и URL, без выдумок.
Один вывод в 5 буллетах со ссылками.
```

✅ `research/sources.md` в проекте → **+ в чат**

В парах. Один — пользователь, один — автор.
