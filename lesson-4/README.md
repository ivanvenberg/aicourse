# Встреча 4 — Автоматизация. Скиллы, агенты и MCP.

## Что сделаем

Научим агента работать с инструментами которыми ты пользуешься каждый день — Figma, Notion, Google Sheets, Gmail. Через MCP агент получает «руки» и начинает делать реальную работу в реальных сервисах.

Плюс создадим скиллы — переиспользуемые команды под твою профессию.

## Структура встречи

**15 минут — фидбек-раунд**
Каждый рассказывает: показал продукт трём людям, вот что они сказали. Один главный инсайт на человека.

**15 минут — что такое MCP**
MCP — это «руки» для Claude. Без MCP он только читает и пишет файлы. С MCP — открывает Figma, редактирует таблицу, отправляет письмо.

**40 минут — подключаем MCP под свою профессию**
Каждый подключает тот MCP который нужен именно ему.

**10 минут — создаём скилл**
Одна команда → один автоматизированный процесс.

**5 минут — домашка**

## MCP под каждую профессию

### 🎬 Моушн дизайнер
**Figma MCP** — Claude читает макеты, комментирует, предлагает правки  
Установка: `claude mcp add figma`  
Что делает: открывает фрейм → описывает что видит → предлагает анимационный сценарий → пишет сторибород

### 🔍 UX researcher
**Notion MCP** — Claude читает и пишет в Notion напрямую  
Установка: `claude mcp add notion`  
Что делает: берёт транскрипт интервью → выделяет инсайты → структурирует в базу знаний в Notion

### 🎨 Дизайнеры
**Figma MCP** — Claude видит макеты и работает с ними  
Установка: `claude mcp add figma`  
Что делает: читает дизайн → пишет бриф для клиента → готовит описание правок

### ✍️ Главреды
**Notion MCP** + **Google Docs MCP**  
Установка: `claude mcp add notion` / `claude mcp add gdrive`  
Что делает: берёт тему → собирает контент-план → пишет черновик в твоём стиле → сохраняет в Notion

### 📊 СМО
**Google Sheets MCP** + **Gmail MCP**  
Установка: `claude mcp add gsheets` / `claude mcp add gmail`  
Что делает: читает таблицу с метриками → анализирует → пишет отчёт → отправляет письмо

### 🏗 Архитектор
**Google Sheets MCP**  
Установка: `claude mcp add gsheets`  
Что делает: берёт параметры объекта → считает смету → записывает в таблицу → готовит резюме для клиента

### 📧 CRM маркетолог
**Google Sheets MCP** + **Gmail MCP**  
Установка: `claude mcp add gsheets` / `claude mcp add gmail`  
Что делает: читает базу → сегментирует → пишет письма под каждый сегмент → отправляет

## Как установить MCP (общая инструкция)

```bash
# В папке проекта:
claude mcp add <название>
```

Claude спросит токен доступа — инструкция по каждому сервису в папке `lesson-4/mcp-guides/`.

Если что-то не устанавливается — не паникуй. Пишешь в чат Claude: *«Помоги установить MCP для Figma, вот что у меня написано»* — и он сам разберётся.

## Что такое скилл

Скилл — это команда которую ты вызываешь одним словом.

Создаётся в папке `.claude/skills/` как `.md` файл:

```
/контент — собери контент-план на неделю из моих тем
/бриф — напиши бриф по описанию проекта
/отчёт — собери недельный отчёт из таблицы
/сторибород — напиши сторибород по брифу
```

Один раз создал — пользуешься всегда.

## Что такое MEMORY.md

Файл который лежит в корне проекта. Claude читает его каждый раз при старте.

Пиши туда: кто ты, как ты работаешь, что важно, что нельзя. Тогда не нужно объяснять контекст каждый раз заново.

Пример для главреда:
```
Я главред медиа о технологиях. Целевая аудитория — менеджеры 30-45 лет.
Тон: умный, но без занудства. Без корпоративного языка.
Всегда проверяй факты перед публикацией.
Контент-план обновляется каждый понедельник.
```
