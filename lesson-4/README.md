# Урок 4 — Запуск продаваемого продукта

**С этого урока — один свой проект до демо.**

## Критерий

Продукт, **который реально можно продать**: конкретному человеку, за конкретную боль, с гипотезой цены.

Не портфолио. Не учебный чат с урока 2 (если только не превращаешь его в платный инструмент).

## Файлы

| Файл | Зачем |
|------|--------|
| `theory.md` | Глава 3 — читай **до** встречи |
| `whattodoo.md` | Что делаем на созвоне |
| `homework.md` | Домашка до урока 5 |
| `brief-template.md` | Скопируй в репо продукта как `brief.md` |
| `ideas-fallback.md` | Нет идеи — возьми строку отсюда |

## Результат урока

- [ ] `brief.md` + цитата «заплатил бы» от человека вне курса
- [ ] Одна ссылка: лендинг-оффер + `/app` с одной платной фичей
- [ ] Скилл `/команда` под твою рутину
- [ ] MCP подключён **или** в домашке
