# Урок 4 — Продукт который можно продать + агенты

**С урока 4 один проект на весь финал курса.**

## Критерий курса

Продукт должен быть таким, что **его реально можно продать** — конкретному человеку за конкретную боль. Не портфолио, не «бот Йода».

## Файлы

| Файл | Что внутри |
|------|-----------|
| `theory.md` | **Глава 3** — читай перед встречей |
| `whattodoo.md` | Задания на встрече |
| `homework.md` | **Домашка после урока 4** |
| `brief-template.md` | Скопируй в свой репо как `brief.md` |
| `ideas-fallback.md` | Нет идеи — возьми отсюда |

## Результат урока

- `brief.md` в репо продукта + цитата «заплатил бы»
- Лендинг-оффер + `/app` в одном деплое
- Одна платная фича работает
- Один скилл + один MCP (или скилл в домашке, если MCP не встал)
