# Урок 4 — Продукт который можно продать + агенты

**С урока 4 один проект на весь финал курса.**

## Критерий курса

Продукт должен быть таким, что **его реально можно продать** — конкретному человеку за конкретную боль. Не портфолио, не «бот Йода».

## Файлы

| Файл | Что внутри |
|------|-----------|
| `theory.md` | **Глава 3:** агенты, MCP, скиллы, тест «можно продать», лендинг+app |
| `whattodoo.md` | Задания на встрече |
| `brief-template.md` | Шаблон `brief.md` для своего репо |
| `ideas-fallback.md` | Заготовки, если нет идеи |

## Результат урока

- `brief.md` в репо продукта + цитата «заплатил бы»
- Лендинг-оффер + `/app` в одном деплое
- Одна платная фича работает
- Один скилл + один MCP (или скилл в домашке, если MCP не встал)

## Домашка

1. Три человека видели лендинг — записал реакции
2. Одна доработка оффера или фичи по их словам
3. Скилл использовал минимум 2 раза на реальной задаче

Принеси: ссылка + цитата + «сколько времени сэкономил скилл».
