# Встреча 4 — Автоматизация. Скиллы и агенты.

## Что сделаем

Научим Claude Code делать сложные вещи одной командой. Создадим скилл — это как кнопка, которую ты нажимаешь раз, и агент делает за тебя целый процесс.

У каждого будет свой агент под свою профессию.

## Структура встречи

**15 минут — разбор фидбека**
Что сказали живые пользователи? Какой инсайт удивил больше всего?

**40 минут — скиллы и агенты**
- Что такое скилл в Claude Code: `/имя` → агент выполняет сценарий
- Создаём скилл под реальную задачу
- Анализ данных: загружаем файл → Claude Code пишет скрипт → получаем инсайты

**30 минут — работа над своим проектом**
Каждый создаёт скилл для своей самой болезненной рутины.

**5 минут — домашка**

## Примеры скиллов по профессиям

**Главред** → `/контент` — агент анализирует тему, предлагает структуру материала, пишет черновик в твоём стиле

**Дизайнер** → `/бриф` — агент задаёт клиенту нужные вопросы и формирует бриф автоматически

**СМО** → `/конкуренты` — агент мониторит нужные источники и собирает дайджест изменений

**Архитектор** → `/расчёт` — агент считает смету по параметрам объекта

**CRM маркетолог** → `/сегмент` — агент анализирует данные и предлагает сегментацию

**Моушн дизайнер** → `/сторибород` — агент генерирует описание сцен по брифу

**UX researcher** → `/интервью` — агент обрабатывает транскрипт и выделяет ключевые инсайты

## Домашка

1. Найди одну задачу которую делаешь руками каждую неделю
2. Создай скилл который делает её за тебя
3. Запусти его на реальных данных
4. Запиши: сколько времени занимала задача раньше и сколько теперь

Принеси цифру на следующую встречу.
