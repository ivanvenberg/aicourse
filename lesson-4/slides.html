4 уровня работы с AI

  • 1. Чат — копируешь сам
  • 2. Редактор — Cursor правит файлы
  • 3. Субагент — подзадача, отчёт тебе

AI-native = уровни 2–4

ReAct

Думаю → Действую → Смотрю результат → снова

Не фантазия — каждый шаг по реальному выводу терминала/файла.

Пример

«Прочитай data.json → напиши скрипт → запусти → сохрани в results/»
