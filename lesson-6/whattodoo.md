# Урок 6 — демо-день

Теория: `theory.md`. Подготовка: `homework.md`.

---

## Часть 1. Демо (60 мин)

**5 мин + 2 мин фидбек** на человека.

Структура — `theory.md` §5.19. Живое демо, не слайды.

Фидбек от зала **как пользователь:**
- Понял зачем?
- Где застрял?
- Что убрать?

---

## 🔥 Code review в парах

Открой репо соседа:

```
Строгий ревьюер. project.md в корне.
1. Хорошо
2. Опасно на демо
3. Три правки (файл + строка)
```

Устно 5 мин. Автор записывает **один** пункт на неделю.

## Часть 3. Закрытие (10 мин)

Каждый одной фразой:

**«До [дата] сделаю ___ для ___ чтобы сэкономить ___ часов в неделю.»**

Имя. Действие. Дата.

## Минимум зачёта

Ссылка + `project.md` + цитата из feedback + живое демо 5 мин.
