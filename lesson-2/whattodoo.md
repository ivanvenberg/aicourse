## Задание 0. Установка Claude Code (все вместе)

```bash
npm install -g @anthropic-ai/claude-code
```

Если npm нет — сначала [nodejs.org](https://nodejs.org) → кнопка LTS → установить.

После установки:
```bash
claude
```

Залогинься через claude.ai → Enter.

Проверка: введи `/usage` — должно показать твоё имя и % использования.

**Полезные команды:**
```
/model    → переключить модель (Opus / Sonnet)
/usage    → сколько лимита использовал
/context  → сколько занято контекстное окно
Escape    → остановить если думает слишком долго
```

✅ Написал + в чат

## 🔥 Упражнение по комнатам: «Найди и почини баг»

Делимся на три команды. Каждая получает сломанное приложение в папке `exercise-bugs/`.

**Команда 🖊️ Бриф** → `brok-brief/`
**Команда 📋 Правки** → `brok-quiz/`
**Команда 💰 Смета** → `brok-tracker/`

**Правила:**
1. Открой приложение в браузере — что-то не работает
2. Нажми F12 → Console — скопируй ошибку
3. Спроси Claude Code: *«Вот ошибка: [ошибка]. Вот что должно происходить: [опиши]. Почини.»*
4. Проверь что работает
5. Задеплой на Vercel
6. Скинь ссылку в общий чат

**Победила команда которая задеплоила первой 🏆**

Подсказка: баги спрятаны в JavaScript. Ищи между тегами `<script>`.
