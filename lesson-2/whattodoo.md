# Что делаем на встрече 2

**Сегодня работаем в парах. Каждая пара к концу встречи деплоит один общий продукт.**

> Пары объявляются в начале встречи. Пиши + в чат когда закончил задание, – если застрял.

---

## Как работать в паре

В каждом задании есть две роли — меняйтесь после каждого задания:

**Навигатор** — думает вслух, формулирует задачу, проверяет результат
**Водитель** — пишет промпты, управляет Claude Code, смотрит в экран

Одна клавиатура. Один экран. Оба думают.

---

## Задание 0. Установка Claude Code (оба)

```bash
npm install -g @anthropic-ai/claude-code
```

После установки:
```bash
claude
```

Залогинься → Enter.

Оба проверяют: `/usage` → видно имя и % лимита.
Оба смотрят: `/context` → % заполнения контекстного окна.
Оба переключают: `/model` → посмотреть какие модели есть.

✅ Оба установили → написали + в чат

## Задание 1. Одна идея на двоих (10 мин)

Каждый называет свою идею — одно предложение:
> «Что я делаю руками каждую неделю — и что меня в этом бесит?»

Выслушали друг друга. Выбрали одну идею — ту что интереснее или полезнее обоим.

Это ваш продукт на следующие пять встреч.

✅ Идея выбрана → написали в чат одним предложением

## Задание 2. PRD голосом (15 мин)

**Навигатор** отвечает на вопросы.
**Водитель** открывает Claude Code, вставляет:

```
Прочитай prdinterview.md и используй инструкции из него.
Задавай вопросы по одному — жди ответа.
После — напиши prd.md и сохрани.
```

Навигатор отвечает голосом или текстом. Водитель не подсказывает — только слушает.

После — меняйтесь. Водитель читает prd.md и говорит что упущено или сформулировано неточно. Вместе дорабатываете.

✅ prd.md готов → написали + в чат

---

## Задание 3. Первая версия продукта (20 мин)

Новый чат. **Водитель** вставляет:

```
Прочитай prd.md.
Создай первую версию продукта — HTML + CSS + JS.
Сохрани в папку product/.
```

Пока работает:
- **Навигатор** смотрит и думает что проверить первым делом
- **Водитель** следит за тем что делает Claude Code

Когда готово — оба открывают `product/index.html` в браузере.

Если что-то не работает:
```
Вот ошибка из консоли (F12): [вставь]
Вот что должно происходить: [опиши]
Почини.
```

✅ Продукт открывается → написали + в чат

## 🔥 Упражнение по комнатам: «Найди и почини баг»

Четыре пары — четыре сломанных приложения в папке `exercise-bugs/`.

Пара получает своё приложение в начале упражнения.

**Порядок:**
1. Открой в браузере — что не работает?
2. F12 → Console → скопируй ошибку
3. Claude Code: «Вот ошибка: [вставь]. Должно происходить: [опиши]. Почини.»
4. Проверь → задеплой на Vercel → скинь ссылку в чат

**Роли в упражнении:**
- Навигатор: читает задание, описывает баг словами, проверяет результат
- Водитель: открывает консоль, работает с Claude Code, деплоит

**Победила пара которая задеплоила первой 🏆**

## Совместная домашка

До следующей субботы пара делает **одно улучшение продукта вместе**:

1. Договоритесь голосом или в переписке — что добавляете
2. Один делает, другой ревьюит результат
3. Задеплойте обновлённую версию
4. Скиньте ссылку + одно предложение: что добавили и зачем

На следующей встрече каждая пара показывает что сделала — 2 минуты.
