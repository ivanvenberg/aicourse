## Задание 0. Установка Claude Code

Открой терминал и введи:

```bash
npm install -g @anthropic-ai/claude-code
```

Потом:
```bash
claude
```

Залогинься через claude.ai → Enter. Готово.

**Проверка:** введи `/usage` — должно появиться твоё имя и процент использования.

✅ Написал плюс в чат

## Задание 1. Твоя идея — одно предложение

Ответь на вопрос вслух или в чат:

> Что я делаю руками каждую неделю — и что меня в этом бесит?

Напиши одно предложение. Например:
- «Каждый понедельник трачу час на составление контент-плана из разных источников»
- «Каждый раз объясняю клиентам одно и то же про правки в дизайне»
- «Вручную собираю отчёт по каналам в конце недели»

Это и есть твой продукт.

✅ Написал плюс в чат

## Задание 2. Пишем PRD голосом

Создай папку для своего проекта. Открой в ней Claude Code.

Скопируй и вставь этот текст:

```
Прочитай файл prdinterview.md и используй инструкции из него.
Задавай мне вопросы по одному, чтобы собрать контекст.
После того как соберёшь достаточно информации — напиши файл prd.md
и сохрани его в эту папку.
```

Отвечай голосом (или текстом) на каждый вопрос.
Не думай долго — говори что первое приходит в голову.

✅ Файл prd.md создан — написал плюс в чат
