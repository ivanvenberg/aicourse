## Задание 0. Установка и знакомство с Claude Code

```bash
npm install -g @anthropic-ai/claude-code
```

После установки:
```bash
claude
```

Залогинься через claude.ai → Enter.

**Пощупай инструмент — теория в действии:**
1. Введи `/usage` — видишь сколько лимита использовал (это токены из главы 1)
2. Введи `/context` — видишь % заполнения контекстного окна
3. Введи `/model` — переключись между Sonnet и Opus (разные модели, разная цена)

Попробуй написать что-нибудь простое:
```
Привет. Ты видишь мою папку? Что в ней есть?
```

✅ Написал + в чат

## Задание 4. Запускаем

Открой `product/index.html` в браузере.

Если что-то не работает — открой консоль (F12 → Console), скопируй ошибку и отдай Claude Code:
```
Вот ошибка из консоли: [вставь]
Вот что должно происходить: [опиши]
Почини.
```

*Это три закона вайбкодинга в действии: конкретно описываешь → проверяешь → итерируешь.*

✅ Продукт работает → написал + в чат

## 🔥 Упражнение по комнатам: «Найди и почини баг»

Три команды. Каждая получает сломанное приложение в папке `exercise-bugs/`.

**Команда 🖊️ Бриф** → `brok-brief/`
**Команда 📋 Правки** → `brok-quiz/`
**Команда 💰 Смета** → `brok-tracker/`

**Порядок действий:**
1. Открой в браузере — что не работает?
2. F12 → Console → скопируй ошибку
3. Claude Code: *«Вот ошибка: [ошибка]. Должно происходить: [опиши]. Почини.»*
4. Проверь
5. Задеплой на Vercel
6. Скинь ссылку в чат
