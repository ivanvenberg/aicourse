# Упражнение: найди и почини баг

## Как это работает

Вы разделились на три команды. Каждая получила сломанное приложение.
Ваша задача: найти баг с помощью Claude Code и починить его.

**Команда 🖊️ Бриф** → папка `brok-brief/`
**Команда 📋 Правки** → папка `brok-quiz/`
**Команда 💰 Смета** → папка `brok-tracker/`

---

## Шаг 1. Откройте приложение

Найдите файл `index.html` в вашей папке.
Откройте его в браузере (двойной клик).
Попробуйте воспользоваться — что-то не работает.

---

## Шаг 2. Откройте консоль

Нажмите **F12** → вкладка **Console**.
Там красным написана ошибка. Скопируйте её.

---

## Шаг 3. Спросите Claude Code

Откройте терминал в папке с вашим приложением.
Напишите Claude Code:

```
Вот ошибка из консоли браузера: [вставьте ошибку]
Вот что должно происходить: [опишите что должна делать кнопка]
Найди и почини баг в index.html
```

---

## Шаг 4. Проверьте

Обновите браузер. Работает?
Если нет — скажи Claude Code что всё ещё не работает.

---

## Шаг 5. Задеплойте

```
Задеплой это приложение на Vercel и дай ссылку
```

Скиньте ссылку в общий чат — победила команда которая задеплоила первой 🏆

---

## Подсказка (только если совсем застряли)

Баги в приложениях спрятаны в JavaScript. Ищите в коде между тегами `<script>`.
Не подсматривайте пока не попробуете сами!
