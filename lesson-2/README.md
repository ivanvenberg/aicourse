# Встреча 2 — Твоя идея. Твой первый продукт.

## Что сделаем

Найдём идею — и сразу начнём её строить. С сегодняшнего дня у каждого есть свой проект, который будет расти все шесть встреч.

Ещё установим Claude Code — инструмент, который пишет код вместо тебя. Ты описываешь словами, он делает.

## Структура встречи

**15 минут — разбор домашки**
Показываем калькуляторы. Что получилось, что сломалось, как чинили.

**40 минут — Claude Code + PRD**
- Устанавливаем Claude Code
- Учимся писать PRD — продуктовые требования словами, без технического языка
- Claude Code собирает первую версию продукта по твоему описанию

**30 минут — работа над своим проектом**
Каждый начинает свой продукт прямо на встрече. Ваня и группа помогают.

**5 минут — домашка**

## Как найти идею (готовься заранее)

Ответь на один вопрос:

> Что ты делаешь руками каждую неделю — и что тебя в этом бесит или отнимает время?

Примеры по профессиям:
- **Главред** → собирать контент-план из разных источников, согласовывать тексты
- **Дизайнер** → писать бриф с нуля каждый раз, объяснять клиенту правки
- **СМО** → вручную мониторить конкурентов, собирать отчёты по каналам
- **Архитектор** → делать расчёты под каждый проект, объяснять клиенту стоимость
- **CRM маркетолог** → сегментировать базу руками, писать письма под каждый сегмент
- **Моушн дизайнер** → писать сторибород с нуля, согласовывать концепт
- **UX researcher** → разбирать транскрипты интервью, выделять инсайты

Принеси на встречу конкретный ответ. Это и будет твой продукт.

## Что нужно установить до встречи

```bash
npm install -g @anthropic-ai/claude-code
```

Если `npm` не установлен — скачай Node.js: [nodejs.org](https://nodejs.org) (кнопка LTS).

После установки запусти:

```bash
claude
```

Залогинься через claude.ai аккаунт — и готово.

Если что-то не получается — не паникуй, разберём в начале встречи.

## Домашка

Возьми то, что начал на встрече. Добавь **одну новую функцию** через Claude Code:

1. Открой терминал в папке проекта
2. Напиши Claude Code что хочешь добавить
3. Задеплой обновлённую версию на Vercel
4. Скинь ссылку в чат

Не знаешь что добавить — спроси Claude Code: *«Посмотри мой проект и предложи три улучшения, которые сделают его полезнее»*.
