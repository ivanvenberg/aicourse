# Встреча 2 — Квиз для друзей

## Что сделаем

Интерактивный квиз — «какой ты персонаж», тест на профессию или викторину. Готовый результат можно будет скинуть друзьям.

## Что узнаем

- Как JavaScript реагирует на действия пользователя
- Как хранить данные (ответы) пока человек проходит тест
- Как показывать разный результат в зависимости от ответов
