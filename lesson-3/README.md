## Что сделаем

Две вещи. Первое — добавим в продукт базу данных, чтобы данные хранились в облаке. Второе — научимся анализировать данные скриптами, а не просить нейронку считать вслепую.

## Структура встречи

**15 минут — фидбек-раунд**
Каждый показывает что добавил за неделю. Что получилось, что сломалось, как чинил.

**20 минут — Supabase: база данных в облаке**
- Регистрируемся, создаём таблицу
- Claude Code подключает базу к продукту
- Данные больше не исчезают после закрытия браузера

**25 минут — анализ данных скриптами**
Важное правило: **нельзя доверять нейронке считать цифры без скрипта**. Она галлюцинирует. Один и тот же файл — разные числа у всех.

Правило простое:
> Хочешь точные цифры → проси Claude написать Python-скрипт → запускай скрипт → доверяй результату

**20 минут — анализируем своё**
Каждый анализирует что-то реальное из своей работы.

**5 минут — домашка**

## Что можно проанализировать — по профессиям

**Моушн дизайнер** → выгрузка из CRM или таблица проектов: сколько проектов в месяц, средний чек, типичные дедлайны

**UX researcher** → транскрипты интервью: Claude пишет скрипт который считает частоту упоминаний тем, выделяет паттерны

**Дизайнеры** → таблица правок от клиентов: сколько итераций в среднем, какие правки повторяются чаще всего

**Главреды** → статистика материалов: просмотры, дочитывания, конверсии — скрипт найдёт что работает лучше

**СМО** → данные по каналам: ROAS, CPL, воронка — скрипт посчитает точно, без галлюцинаций

**Архитектор** → таблица проектов: какие объекты самые маржинальные, где теряешь время

**CRM маркетолог** → база подписчиков: отток по месяцам, сегменты по активности, LTV

## Исследование конкурентов (задание на урок)

Claude Code умеет читать сайты и делать анализ конкурентов.

Промпт:
```
Прочитай мой PRD (файл prd.md).
Найди 5-7 конкурентов моего продукта на российском рынке.
Для каждого: читай их сайт, ищи отзывы.
Составь сравнительную матрицу.
Найди мои точки отстройки.
Сохрани в competitor-research.md
```

Важно: просить давать **ссылки на источники** — иначе будут галлюцинации.
