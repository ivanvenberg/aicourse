## Что такое база данных

База данных — это место где данные хранятся постоянно, не исчезают когда закрываешь браузер, и доступны с любого устройства.

**Без базы данных:**
Пользователь добавил задачу → закрыл браузер → задача исчезла. Данные жили только в памяти браузера.

**С базой данных:**
Пользователь добавил задачу → данные сохранились в облаке → открыл на телефоне → всё на месте.

---

## Что такое Supabase

Supabase — это готовая база данных в облаке. Бесплатно. Не нужен свой сервер.

Ты регистрируешься → создаёшь таблицу (как в Excel) → Claude Code пишет код который читает и записывает данные.

**Почему Supabase, а не что-то другое:**
- Бесплатный план достаточен для наших проектов
- Claude Code умеет с ним работать из коробки
- Есть понятный интерфейс чтобы смотреть что лежит в базе

---

## Почему нельзя доверять LLM считать цифры

Это главный инсайт урока. Запомни:

> Один и тот же файл с данными → разные люди получают разные цифры от нейронки.

Пример: спрашиваем NPS из одного файла reviews.json.
- У одного: NPS = 36
- У другого: NPS = 24
- У третьего: NPS = 41

Данные одни. Нейронка галлюцинирует.

**Почему так происходит:**
LLM — это языковая модель. Она предсказывает слова. Она не умеет считать — она угадывает правдоподобный ответ.

**Решение:**
Проси Claude Code написать Python-скрипт → он считает детерминированно → у всех одинаковый результат.

---

## Что такое Python-скрипт

Скрипт — это небольшая программа которая делает одну конкретную задачу.

Тебе не нужно знать Python. Claude Code напишет скрипт за тебя.

Ты говоришь: «Напиши скрипт который посчитает средний NPS из файла reviews.json»
Claude Code пишет скрипт → запускает его → выдаёт точный результат.

**Для каких задач нужны скрипты:**
- Считать конверсии в воронке
- Считать NPS, средний чек, LTV
- Анализировать когорты
- Сегментировать базу пользователей
- Любые точные числовые расчёты

## ABCD-сегментация

Простая и мощная сегментация клиентов по двум осям: маржинальность + удовлетворённость.

| Сегмент | Кто это | Что делать |
|---------|---------|-----------|
| A | Доволен + платит много | Беречь, развивать |
| B | Доволен + платит меньше | Растить до A |
| C | Недоволен, около нуля | Разобраться почему |
| D | Недоволен + не платит | Не тратить время |

Claude Code может сделать ABCD-сегментацию из любой таблицы с клиентами.

## Анализ конкурентов через Claude Code

Claude Code умеет открывать сайты, читать страницы с отзывами, и делать сравнительную матрицу.

**Промпт:**
```
Прочитай мой prd.md.
Найди 5-7 конкурентов на российском рынке.
Для каждого: прочитай сайт, найди отзывы.
Сравнительная матрица: функции, цены, позиционирование.
Найди мои точки отстройки.
Приводи ссылки на источники.
```

**Важно:** чем больше ссылок на источники — тем меньше галлюцинаций.

## Как визуализировать данные

После анализа можно попросить красивый отчёт:

```
На основе всех файлов из папки results/
создай HTML-страницу с визуализацией данных.
Добавь графики, ключевые цифры, рекомендации.
```

Claude Code создаёт HTML который можно открыть в браузере или отправить руководителю.
