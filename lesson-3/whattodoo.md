## 🔥 Упражнение по комнатам: «Докажи что AI врёт»

Это главный эксперимент встречи. Делитесь на три группы.

Каждая группа работает с одним и тем же файлом `data/reviews.json`.

**Каждый делает одно и то же:**
```
Прочитай файл data/reviews.json.
Посчитай NPS: (% оценок 9-10) - (% оценок 0-6).
Ответь одним числом.
```

Запишите результат и скиньте в общий чат.

**Смотрим: у всех разные числа? Да. Почему?**

Потому что AI не считает — он угадывает правдоподобный ответ.

**Теперь исправляем:**

Новый чат. Вставь:
```
Напиши Python-скрипт который читает data/reviews.json и считает NPS.
Формула: ((кол-во оценок 9-10) / всего) - ((кол-во оценок 0-6) / всего) × 100
Запусти и выведи результат.
Сохрани скрипт в scripts/nps.py
```

Скиньте результат в чат. Сравниваем: теперь у всех одно число? ✅

**Вывод который запомнится навсегда:** хочешь точные числа → проси скрипт.
