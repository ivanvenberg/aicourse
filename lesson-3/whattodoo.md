## Задание 1. Эксперимент с промптами (10 мин)

*Практика из главы 2: zero-shot vs few-shot vs chain-of-thought.*

Возьми любую задачу связанную с твоим продуктом. Напиши три версии промпта:

**Zero-shot:** просто попроси
```
Придумай название для моего инструмента который [что делает]
```

**Few-shot:** дай примеры нужного формата
```
Примеры хороших названий SaaS-продуктов:
- Notion → кратко, можно угадать смысл
- Linear → одно слово, ощущение скорости
- Loom → звучит тепло

Придумай название для инструмента который [что делает]
```

**Chain-of-thought:** попроси рассуждать
```
Придумай название для инструмента который [что делает].
Сначала выдели 3 ключевых ощущения которые он должен передавать.
Потом предложи 5 вариантов названий. Потом выбери лучшее и объясни почему.
```

Сравни результаты. Какой подход дал лучший результат для твоей задачи?

✅ Написал наблюдение в чат

## 🔥 Упражнение по комнатам: «Докажи что AI врёт»

*Практика из главы 2: почему LLM не умеет считать.*

**Все читают один файл `lesson-3/data/reviews.json`.**

Каждый делает одно и то же:
```
Прочитай файл data/reviews.json.
Посчитай NPS: (% оценок 9-10) - (% оценок 0-6).
Ответь одним числом.
```

Скинь результат в чат. Смотрим: у всех разные числа? Почему?

**Теперь исправляем:**
```
Напиши Python-скрипт который читает data/reviews.json и считает NPS.
Формула: (кол-во 9-10 / всего - кол-во 0-6 / всего) × 100
Запусти и выведи результат.
```

Скинь результат. У всех одинаково? Вот почему нужен скрипт.

✅ Написал + в чат
