## Ключевые решения формата

1. **Работа между встречами**, не только на созвоне
2. **Show-and-tell 5 мин:** что сделала → затыки → куда иду
3. **project.md на GitHub** у каждого (приватный repo — ок)
4. **Skills ≤50 строк**, несколько маленьких rules
5. **3 агента-персоны** челленджат идею до кодинга → `agents/verdict.md`
6. **Парсинг не врёт** — общий принцип для Юли, Иры, Димы

## Теория на созвоне (кратко)

- **Skills** — повторяемые инструкции по тегу; не один жирный rule
- **Parallel / Perplexity** — поиск с citations, не «спроси ChatGPT»
- **Параллельные агенты** — разные персоны «ругаются», лучше анализ
- **Контекстное окно** — держать минимальным; RAG top-3, не вся книга
- **Email/дизайн в LLM** — дорого по токенам; готовые сервисы (Slava Bell, Framer)
