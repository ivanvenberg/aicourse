# Лера — исследования (skills, не «ещё один ChatGPT»)

## Зачем это тебе

Бриф → план исследования → синтез → слайды — ты уже делаешь с ChatGPT.  
Vibe-coding здесь = **повторяемый пайплайн**: один раз настроила правила → каждый раз одинаково, **без** загрузки сырых интервью (NDA / Сбер).

Транскрипция сессий психолога — **не этот проект** (есть Granola и аналоги).

---

## К чему идём (финал к 13 июня)

Набор **коротких инструкций** (skills) + шаблоны md: бриф → plan → (потом) синтез → outline слайдов.  
Demo на **выдуманных** данных. Repo может быть **приватным**.

---

### 3. Первый skill (~1,5 ч)

Файл `.cursor/rules/research-brief.mdc` (до 50 строк):

- **Вход:** текст брифа (можно выдуманный)
- **Выход:** `output/plan.md` — цели, метод, вопросы, сроки
- В конце plan: «**проверить вручную**»

**Готово, когда:** прогнала **тестовый** бриф → plan за пару минут.
