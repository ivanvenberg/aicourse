## Apify (общее для парсинга)

**[Apify](https://apify.com/pricing)** — free plan: **~$5 usage credits каждый месяц**, **без карты**. Кредиты не копятся; если кончились — пауза до нового цикла (без привязанной карты **нет** неожиданных списаний).

**Кому:** Юля (Telegram), Ира (если MCP застрял), Дима (студии / контакты).

**Правило курса:** неделя 1 = **0 ₽ и ручная проверка**; Apify — **ускоритель после** manual-check, не замена «не врёт».
