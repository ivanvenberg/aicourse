# Шаблон manual-check — «парсинг не врёт»

Общий для **Иры**, **Юли**, **Димы** (outreach). Скопируй в свой проект: `tests/manual-check.md`.

---

## Как пользоваться

1. Выбери **10** записей (продукты / лиды / студии).
2. Проверь **руками** в источнике (сайт, чат, contact page).
3. Сравни с выводом скрипта.
4. На созвоне покажи **эту таблицу** — 2 минуты, zero bullshit.

## Правила

| Правило | Пример |
|---------|--------|
| Нет URL → не ✅ | ВкусVill: нет `source_url` = не «в наличии» |
| Скрипт `null` / ❓ — норм | Лучше честно, чем галлюцинация |
| Одна строка «❌ не совпало» — ценнее 10 «✅» | Показывает, что ты проверяешь |

---

## Для Иры (ВкусVill)

- **L1:** товар в каталоге (MCP / search)
- **L2:** заказ по адресу — отдельная колонка «корзина руками», если API не даёт

## Для Юли (лиды)

- Колонка «источник» = ссылка на **конкретное сообщение**

## Для Димы (студии)

- Колонка «email» = только если видел на **contact page** (URL в строке)
