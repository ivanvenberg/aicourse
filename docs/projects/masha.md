# Маша — todo + pixel-сад

## Зачем это тебе

Обычные планировщики **надоедают**. Тут задача → сделала → **что-то растёт на карте** — хочется вернуться. Красота мотивирует; код на первой неделе может быть **простым и некрасивым** — это ок.

---

## К чему идём (финал к 13 июня)

Сайт по ссылке: зашла → задачи → отметила done → **посадила декор** → перезагрузила — всё на месте.  
Потом — красивее art, может друзья. **Не сейчас.**

## План на эту неделю

Один путь, без развилок: **localStorage** (данные в браузере). База данных — **потом**, если захочешь.

### 1. project.md (~30 мин)

- Одна фича: «задача выполнена → предмет на карте»
- Скопируй [`project-template.md`](../../lesson-4/project-template.md)

**Готово, когда:** в `project.md` одна фича, без друзей и multiplayer в scope этой недели.

---

### 2. Каркас приложения (~2–3 ч)

Попроси Cursor:

```
Прочитай project.md.
Next.js: /login (ник + выбор из 4 аватаров), /tasks, /garden (сетка 16×16).
Отметила задачу done → выбрала клетку → сохранилось в localStorage.
Плитки — простые квадраты 16×16, не идеальный pixel art.
README: npm install && npm run dev
```

**Готово, когда:** на localhost проходишь цикл: задача → done → клетка → **F5** → клетка осталась.

---

### 3. schema.md (~15 мин)

Коротко опиши: пользователь, задачи, клетки `{x, y, sprite}`.

---

### 4. Деплой (~45 мин)

**Готово, когда:** та же ссылка работает не только у тебя на ноуте.
