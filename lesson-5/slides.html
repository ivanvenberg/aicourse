RAG — 7 шагов

Доки → чанки → эмбеддинги → хранилище
Вопрос → эмбеддинг → top-3 чанка
Чанки + вопрос → модель → ответ

Старт: один knowledge.md + «не знаю»

Почему RAG ломается

  • Плохой чанкинг → overlap 50 слов
  • Устаревшие данные → даты в metadata
  • Слишком много чанков → top-3 + rerank
  • Нет «не знаю» → галлюцинации

Эмбеддинги + pgvector

Текст → вектор ~1536 чисел

Похожий смысл → близкие векторы
