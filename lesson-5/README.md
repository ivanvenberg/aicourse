# Встреча 5 — Продукт становится умным. Telegram-бот с AI.

## Что сделаем

Подключим к продукту настоящий AI — через Anthropic API. И сделаем Telegram-бота который работает и отвечает сам.

## Структура встречи

**15 минут — разбор скиллов**
Каждый рассказывает: что автоматизировал, сколько времени сэкономил. Считаем вместе.

**40 минут — Anthropic API + Telegram-бот**
- Как работает API: ты отправляешь запрос → Claude отвечает → бот передаёт пользователю
- Регистрируем бота через @BotFather
- Claude Code пишет и деплоит бота — за один сеанс

**30 минут — работа над своим проектом**
Каждый выбирает: добавить AI в существующий продукт или сделать отдельного бота.

**5 минут — домашка**

## Что можно сделать

**Главред** → бот который редактирует тексты в твоём стиле и по твоей редполитике

**Дизайнер** → бот который принимает бриф от клиента и задаёт нужные вопросы

**СМО** → бот который готовит еженедельный дайджест по твоим каналам

**Архитектор** → бот для клиентов который отвечает на типовые вопросы по проекту

**CRM маркетолог** → бот который генерирует письма под сегмент по одной команде

**Моушн дизайнер** → бот который превращает бриф в детальный сторибород

**UX researcher** → бот который обрабатывает транскрипт интервью и выдаёт инсайты

## Что нужно подготовить

- Аккаунт на [console.anthropic.com](https://console.anthropic.com) — зарегистрируйся заранее
- Telegram на телефоне — он точно уже есть

## Домашка

1. Запусти бота
2. Поделись с **пятью людьми** — попроси реально попользоваться
3. Собери: что попросили добавить? Что сломалось?
4. Сделай хотя бы одно улучшение на основе их слов

На демо-дне будет видно кто итерировал, а кто просто задеплоил.
