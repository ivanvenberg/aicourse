# Встреча 5 — Telegram-бот с AI + контент-пайплайн

## Что сделаем

Создадим Telegram-бота который отвечает с помощью AI. И соберём контент-пайплайн — автоматическую систему которая сама находит новости и пишет посты в твоём стиле.

## Структура встречи

**15 минут — разбор скиллов**
Каждый рассказывает: что автоматизировал, сколько времени сэкономил. Считаем вместе.

**20 минут — Telegram-бот**
- Регистрируем бота через @BotFather — получаем токен
- Подключаем OpenRouter (агрегатор всех AI моделей — дешевле чем напрямую)
- Claude Code пишет и запускает бота
- Тестируем: пишем боту — он отвечает

**30 минут — контент-пайплайн (особенно для главредов)**

Система из трёх скиллов:

**Скилл 1: tone-of-voice**
Субагент читает все твои тексты → выделяет стиль → сохраняет как скилл

**Скилл 2: topic-map**
Субагент читает твои материалы → строит карту тем → что растёт, что не пишешь, стоп-лист

**Скилл 3: find-news**
Агент открывает браузер → ходит по нужным сайтам → фильтрует по твоим темам → список актуального

**Скилл 4: create-posts**
Берёт новости → для каждой запускает агента-копирайтера → пишет пост в твоём стиле

Результат: одна команда `/посты` → готовые черновики постов на сегодня.

**10 минут — домашка**

## Что такое OpenRouter

Агрегатор AI моделей. Регистрируешься один раз — получаешь доступ к Claude, Gemini, GPT, Mistral и другим. Наценка +5% от цены провайдера.

Для бота поддержки выгоднее использовать Gemini Flash через OpenRouter — дешевле, достаточно умный для ответов на типовые вопросы.

Регистрация: [openrouter.ai](https://openrouter.ai)

## Как хранятся скиллы

```
.claude/
  skills/
    tone-of-voice/
      skill.md    ← инструкции для агента
    find-news/
      skill.md
    create-posts/
      skill.md
```

Вызов через `/tone-of-voice` или `/find-news` в Claude Code.

## Примеры ботов по профессиям

**Главред** → бот редактирует тексты в стиле редполитики, отвечает на вопросы авторов

**Дизайнер** → бот принимает бриф от клиента, задаёт правильные вопросы, структурирует

**СМО** → еженедельный дайджест по каналам по запросу

**Архитектор** → бот отвечает клиентам на типовые вопросы по проекту

**CRM маркетолог** → бот генерирует письма под сегмент по одной команде

**Моушн дизайнер** → бот превращает бриф в детальный сторибород

**UX researcher** → бот обрабатывает транскрипт и выдаёт топ инсайтов

## Домашка

1. Запусти бота
2. Поделись с пятью людьми — попроси реально попользоваться
3. Запиши: что попросили добавить, что сломалось
4. Сделай хотя бы одно улучшение на основе их слов

На демо-дне будет видно кто итерировал, а кто просто задеплоил.
