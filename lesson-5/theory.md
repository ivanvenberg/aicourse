## Что такое API

API — это способ для программ общаривать между собой.

Когда ты пишешь Telegram-боту:
1. Telegram получает твоё сообщение
2. Передаёт его твоей программе
3. Твоя программа отправляет запрос к AI (Claude / Gemini)
4. AI отвечает
5. Твоя программа отправляет ответ обратно в Telegram
6. Ты видишь ответ в чате

Ты не видишь эту цепочку — ты просто пишешь боту и получаешь ответ.

## OpenRouter: один ключ ко всем моделям

Из книги «Vibe Coding»:
> «Рынок AI моделей меняется каждую неделю. Вместо того чтобы разбираться с каждым провайдером отдельно, используй агрегатор.»

**OpenRouter** — сервис который даёт доступ к Claude, Gemini, GPT, Mistral и другим через один API.

**Почему удобно:**
- Одна регистрация → все модели
- Легко переключаться между моделями
- Наценка всего +5% от цены провайдера

**Для бота поддержки:**
Gemini Flash через OpenRouter — быстрый, дешёвый, достаточно умный для ответов на типовые вопросы. В 5-10 раз дешевле чем Claude Opus.

Регистрация: [openrouter.ai](https://openrouter.ai)

## Как работает Telegram-бот

**Шаг 1:** Регистрируешь бота через @BotFather → получаешь токен
**Шаг 2:** Твой код запускается (локально или на сервере)
**Шаг 3:** Бот слушает входящие сообщения (метод polling)
**Шаг 4:** При получении → отправляет в AI → получает ответ → отправляет обратно

**Токен бота:** уникальный ключ вида `123456:ABCdef...`. Никому не показывай — с ним можно управлять твоим ботом.

## RAG: база знаний для бота

**RAG (Retrieval-Augmented Generation)** — метод когда AI отвечает не из своей голову, а на основе конкретных документов которые ты ему дал.

**Без RAG:**
> «Расскажи о политике возврата» → AI придумывает правдоподобный ответ

**С RAG:**
> AI читает файл с реальной политикой возврата → отвечает точно

**Как сделать:**
```
Добавь в бота базу знаний из файла faq.md.
Бот должен отвечать только на основе этого файла.
Если вопрос выходит за пределы базы — честно говорить что не знает.
```

## Оптимизация: когда использовать какую модель

Из книги «Vibe Coding»:
> «Разные задачи требуют разных инструментов. Как в кухне: для тонкой работы нужен маленький нож, для рубки — тесак.»

| Задача | Модель | Почему |
|--------|--------|--------|
| Ответы на типовые вопросы | Gemini Flash | Быстро и дёшево |
| Написание постов в стиле автора | Claude Sonnet | Качественный текст |
| Сложный анализ | Claude Opus | Лучшее рассуждение |
| Простые скрипты | Sonnet / Gemini | Дёшево |

Переключать модель в Claude Code: `/model`

## Безопасность

После запуска бота — обязательно:

```
Проверь безопасность моего бота:
- нет ли API ключей в коде (они должны быть в .env файле)
- правильно ли настроены права доступа
- есть ли защита от спама
Исправь если найдёшь проблемы.
```

Из книги: один разработчик в книге не проверил безопасность — его сервер взломали и использовали для майнинга криптовалюты.
