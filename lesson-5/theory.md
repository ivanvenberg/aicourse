## Что такое API

API — это способ для программ общаться между собой.

Когда ты пишешь Telegram-боту → Telegram передаёт сообщение твоей программе → программа отправляет запрос к AI → AI отвечает → программа отправляет ответ обратно в Telegram.

Ты не видишь эту цепочку — ты просто пишешь боту и получаешь ответ.

---

## OpenRouter

OpenRouter — агрегатор AI моделей. Одна регистрация → доступ к Claude, Gemini, GPT, Mistral и другим.

**Почему OpenRouter, а не напрямую Anthropic:**
- Дешевле для маленьких объёмов
- Легко переключать модели
- Для бота поддержки можно использовать Gemini Flash — быстрее и дешевле

**Сколько стоит:**
- Регистрация бесплатная
- Пополняешь баланс от $5
- Наценка +5% сверху цены провайдера

Регистрация: [openrouter.ai](https://openrouter.ai)

## Как работает Telegram-бот

1. Создаёшь бота через @BotFather → получаешь токен
2. Твой код запускается на сервере (или локально)
3. Бот слушает входящие сообщения (polling или webhook)
4. При получении сообщения — отправляет его в AI
5. Ответ AI → отправляет обратно пользователю

**Что такое токен бота:**
Это уникальный ключ который идентифицирует твоего бота. Храни его в секрете — с ним можно управлять ботом.

**Polling vs Webhook:**
- Polling: бот каждые несколько секунд спрашивает «есть новые сообщения?» — проще для разработки
- Webhook: Telegram сам отправляет сообщения на твой адрес — нужен для production

Claude Code настроит polling — этого достаточно для нашего курса.

## Что можно сделать с ботом

**Бот поддержки:** отвечает на типовые вопросы по базе знаний которую ты ему дал.

**Бот-редактор:** принимает текст, правит по твоей редполитике, возвращает исправленную версию.

**Бот-бриф:** задаёт клиенту правильные вопросы и формирует бриф.

**Бот-дайджест:** по запросу собирает что произошло за неделю.

**Бот-сторибород:** принимает бриф, возвращает детальный сторибород.
