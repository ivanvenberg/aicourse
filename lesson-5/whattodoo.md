## Задание 0. Регистрируемся в OpenRouter

Зайди на [openrouter.ai](https://openrouter.ai)
Зарегистрируйся → Keys → Create Key → скопируй ключ.

✅ Ключ есть — написал плюс в чат

---

## Задание 1. Создаём бота в Telegram

Открой Telegram → найди @BotFather → напиши `/newbot`

Следуй инструкциям:
1. Придумай имя бота (например: «Мой помощник»)
2. Придумай username — должен заканчиваться на `bot` (например: `my_helper_2025_bot`)
3. Скопируй токен — он выглядит как `123456:ABCdef...`

✅ Токен есть — написал плюс в чат

---

## Задание 2. Создаём бота

Создай папку `telegram-bot/`. Открой в ней Claude Code.

Вставь (подставь свои данные):

```
Создай Telegram-бота поддержки.

Токен бота: [вставь токен от BotFather]
API ключ OpenRouter: [вставь ключ]
Модель: google/gemini-flash-1.5

Бот должен:
- Принимать текстовые сообщения
- Отвечать на вопросы о [опиши о чём будет бот — твой продукт, твоя работа]
- Быть вежливым и лаконичным

Создай файл bot.py, requirements.txt и инструкцию как запустить.
Запусти бота локально.
```

✅ Бот запущен — написал плюс в чат

---

## Задание 3. Тестируем бота

Найди своего бота в Telegram (по username который придумал).
Напиши ему что-нибудь. Он должен ответить.

Если не отвечает — скажи Claude Code:
```
Бот не отвечает на сообщения. Посмотри логи и почини.
```

✅ Бот отвечает — написал плюс в чат

---

## Задание 4. Даём боту знания (RAG)

Добавь боту файл с информацией — чтобы он отвечал на основе реального контекста, а не выдумывал:

```
Добавь в бота базу знаний из файла [путь к файлу с информацией о продукте или FAQ].
Бот должен отвечать основываясь на этой информации.
Если вопрос выходит за пределы базы — честно говорить что не знает.
```

✅ Бот отвечает по базе знаний — написал плюс в чат

## Задание 5. Контент-пайплайн (для главредов)

Если ты главред или создаёшь контент — собираем пайплайн:

```
Создай скилл find-news который:
- Открывает через Chrome Extension новостные сайты: [твои источники]
- Фильтрует новости по темам из моей карты тем
- Сохраняет список в news-today.md

После этого запусти агентов-копирайтеров:
- Для каждой новости — один агент пишет пост в моём стиле (tone-of-voice)
- Сохраняет в posts/[дата]/
```

✅ Посты готовы — написал плюс в чат
