# Встреча 1 — Твой сайт в интернете

## Что сделаем

Личную страницу, которую можно отправить маме по ссылке.

## Что узнаем

- Как устроен HTML (теги, структура)
- Как работает CSS (цвета, шрифты, отступы)
- Как опубликовать сайт бесплатно (Netlify / GitHub Pages)

## Файлы

- `index.html` — стартовый шаблон, с которого начнём

## Задание до встречи

Открой `index.html` в браузере (просто дважды кликни на файл) и посмотри что получилось. Больше ничего делать не нужно — всё остальное на встрече!
