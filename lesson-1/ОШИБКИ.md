# Ошибки и задания — только для тебя

## 🌵 кактус (магазин кактусов)

**Ошибка в script.js:**
`document.querySelectorAll("add-btn")` — пропущена точка перед классом
Должно быть: `document.querySelectorAll(".add-btn")`
Из-за этого кнопки "В корзину" вообще не реагируют на нажатие.

**Задание изменить:**
- Название растения месяца: "Цереус гигантский" → "Агава голубая" (в index.html, id="plantName")
- Цену: ₽2 400 → ₽3 800 (в index.html, id="plantPrice")

---

## 🧛 вампир (ночной клуб)

**Ошибка в style.css:**
`.event-section { display: none; }` — вся секция с событием скрыта
Должно быть: `display: block;` или удалить эту строку
Из-за этого на странице не видно ни название события, ни цену билета, ни кнопку.

**Задание изменить:**
- Дату события: "13 июня 2025" → "31 октября 2025" (в index.html, id="eventDate")
- Цену билета: ₽2 000 → ₽3 500 (в index.html, id="ticketPrice")

---

## 🐟 щука (рыбный ресторан)

**Ошибка в script.js:**
`document.getElementById("reserveBtn")` — неправильный id
Должно быть: `document.getElementById("resBtn")`
Из-за этого кнопка "Забронировать" не работает — страница падает с ошибкой.

**Задание изменить:**
- Название улова дня: "Дорада на гриле" → "Сибас с травами" (в index.html, id="catchName")
- Часы работы в header: "Пн–Пт: 12:00–23:00 · Сб–Вс: 11:00–00:00" → "Ежедневно: 11:00–01:00" (в index.html, класс .hours)

5. Сделать изменения из задания
6. Зарегистрироваться на vercel.com
7. Задеплоить папку — получить живую ссылку
