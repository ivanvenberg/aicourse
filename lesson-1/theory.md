**Результат урока:** страница в браузере + ссылка, которую можно отправить кому угодно.

## 0.1 Что такое вайбкодинг

**Вайбкодинг** — ты описываешь задачу словами, а AI пишет и правит код в файлах.

Ты не учишь синтаксис наизусть. Ты:
- формулируешь что нужно;
- смотришь результат;
- говоришь что поправить.

Это не «AI сделает всё сам». Это **ты рулёшь, AI крутит педали**.

---

## 0.2 Чат в браузере vs агент в Cursor

| | ChatGPT / Claude в браузере | Cursor + агент |
|--|----------------------------|----------------|
| Где живёт код | В чате, копипаст | В файлах проекта |
| Запуск | Ты сам | Агент может запустить |
| Деплой | Нет | `git push` → живая ссылка |
| Память | Один чат | Файлы + git-история |

**Правило курса:** продукт = папка с файлами + репозиторий + деплой. Не скриншот из чата.

---

## 0.3 HTML, CSS, JavaScript — три роли

**HTML** — структура. Что на странице: заголовок, текст, кнопка, форма.

**CSS** — вид. Цвета, шрифты, отступы, «чтобы не выглядело как Word 97».

**JavaScript** — поведение. Кликнул кнопку → что-то произошло. Отправил форму → ушло на сервер.

Когда просишь агента — можно говорить по-русски: *«сделай кнопку синей»* (CSS), *«при клике покажи сообщение»* (JS).

## 0.4 Как сайт оказывается в интернете

Упрощённая цепочка:

```
Файлы на компе → git push → GitHub (хранилище)
                                    ↓
                              Vercel / Netlify (хостинг)
                                    ↓
                              Ссылка https://...
```

**GitHub** — облачная копия кода + история изменений.  
**Vercel / Netlify** — сервис, который берёт код и выкладывает сайт.  
**Деплой** — момент, когда по ссылке открывается новая версия.

---

## 0.5 Git — зачем, не только команды

**Репозиторий** — папка проекта под контролем git.  
**Коммит** — снимок «вот так было в этот момент». Сломалось → можно откатиться.  
**push** — отправить коммиты на GitHub.  
**pull** — забрать обновления курса (`git pull` перед каждой встречей).

AI может удалить файл без спроса. Git — страховка. Коммить перед большими правками.
