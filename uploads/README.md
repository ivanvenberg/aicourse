## uploads/ — куда ребята загружают задания

Эта папка для того, чтобы складывать сюда файлы по встречам: домашки, ссылки, скрины, материалы.

### Как складывать, чтобы всем было удобно

- **Структура**: `uploads/<встреча-N>/<имя-фамилия>/...`
- **Пример**: `uploads/meeting-1/lera-kryuchkova/index.html`

### Как отправлять изменения в репозиторий

Самый безопасный способ — через Pull Request (PR): так изменения не ломают общий код и их легко принять.

Если ты не знаешь, что выбрать — просто напиши Ване, и он подскажет самый быстрый путь.
